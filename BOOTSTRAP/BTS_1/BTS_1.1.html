A cosa serve bootstrap?

Bootstrap è una raccolta di strumenti basati su html e css per la progettazione di siti web reattivi

Cosa devi importare per creare una pagina web con bootstrap

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

1

Migliora il tempo di caricamento del sito Web - Poiché i file javascript e i fogli di stile di Bootstrap verranno caricati da un server vicino in base alla posizione geografica del visitatore del sito Web, il tempo di caricamento sarà ridotto. Immagina un visitatore dal Giappone: se i tuoi file Bootstrap sono ospitati negli Stati Uniti, il tempo di caricamento sarà molto più lento che se avessi una CDN con server installati anche in Asia;

2

Costo della larghezza di banda ridotto - Poiché Bootstrap CDN è già utilizzato da oltre 8 milioni di siti Web, è probabile che una percentuale significativa del traffico del tuo sito Web contenga i file Bootstrap nella cache del browser locale, rendendo il tuo sito Web ancora più veloce;

3

Sicurezza del sito Web migliorata : la distribuzione di contenuti tramite CDN può proteggere meglio il tuo sito Web dalla mitigazione degli attacchi DDoS;

A cosa serve questo attributo del tag meta?

The width=device-width part sets the width of the page to follow the screen-width of the device (which will vary depending on the device).
The initial-scale=1 part sets the initial zoom level when the page is first loaded by the browser.

Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)

La .container classe fornisce un contenitore reattivo a larghezza fissa
La .container-fluid classe fornisce un contenitore a larghezza intera , che copre l'intera larghezza della finestra
